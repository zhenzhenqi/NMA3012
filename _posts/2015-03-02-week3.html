Think of each element on your page as a football player in a group photo shoot. Click on this link and resize the page, notice what's happening to the football players.http://alistapart.com/article/responsive-web-design

This, is exactly what Bootstrap Grid System does for you.

Now, let's make some rows and columns to build our first responsive page on index.html.

Bootstap has already has a very good set of demos online. Click here to go to Boostrap's offical documentation.

Bootstrap scaffolding has three basic elements, Container, Row, and Column. Whenever you start a new page, create a new Container insider Body Tag. Then, puts rows right inside the container, and Columns inside each Row.

Bootstrap already has designed some basic mediaquery settings for us, through four Column classes, col-xs, col-sm, col-md, col-lg.

For beginners, we can simply remember that any column design based on col-xs class will be rendered on cellphones, designs based on col-sm class will be rendered on tablets, designs based on col-md and col-lg will be rendered based on desktops.

We will use their code to do some execrise. But before that, let's add some stying code so we can see every DIV clearly.

Copy and paste the code below inside your custom.css file.

{% highlight CSS %} div { border:1px solid rgba(0,0,0,0.1); background: #9C9C9C; text-align:center; } .container-fluid, .container, .row{ background: #525252; } {% endhighlight %}
