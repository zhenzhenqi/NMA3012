Object Oriented Programming

Make a simple car drive off from the right end of the screen.

{% highlight js %} color c; float xpos; float ypos; float xspeed; void setup() { rectMode(CENTER); size(480, 270); c = color(175); xpos = width/2; ypos = height/2; xspeed = 1; } void draw() { background(255); stroke(0); fill(c); rect(xpos, ypos, 20, 10); xpos = xpos + xspeed; if (xpos > width) { xpos = 0; } } {% end highlight%}
